Tómbola electrónica

Stack: MEAN

Proyecto realizado para la empresa de telecomunicaciones COORPOCENTRO con la finalidad de tener una base de datos de personas que son fieles televidentes al canal 9 (VTV), con esto poder realizar un sorteo en vivo durante la transmisión de los programas que tiene el canal, se muestra el numero de teléfono ganador en la parte inferior de la pantalla.

Calendario de reservaciones

Stack: MEAN

Proyecto realizado para la empresa de telecomunicaciones COORPOCENTRO con la finalidad de tener un control de las horas de uso de los estudios de grabación, con esto se hace una reservasión indicando fecha y hora, para tener un control y evitar choques de horarios de grabación.

Test de preguntas anime (Juego)

Stack: MERT

Proyecto realizado para expandir los conocimientos en desarrollo frontend y backend, con TypeScript se configuró un Rest API para el contenido de las preguntas y respuestas, se consumió ese servicio con en el framework ReactJS o libreria para muchos, se utilizó una base de datos en la nube con mongoDb atlas. La aplicación se utiliza en el programa de Tv. PowerUp, para regalar premios al mas conocedor sobre el anime y manga.
